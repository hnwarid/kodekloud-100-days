# 100 Days of KodeKloud

This repository documents my journey through the 100 Days of KodeKloud challenge, focusing on the **DevOps** and **MLOps** learning paths.

## 🚀 The Goal
To consistently solve real-world infrastructure, automation, and machine learning operations tasks every day for 100 days, ending around late October 2026.

## 📂 Repository Structure
- **`/DevOps`**: Scripts and solutions for system administration, Linux fundamentals, CI/CD, containerization, and infrastructure tasks.
- **`/MLOps`**: Scripts and solutions for machine learning environments, data pipelines, model serving, and deployment tasks.
